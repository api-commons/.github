# API Commons

Open, machine-readable API operations — openly governed under
[apicommons.org](https://apicommons.org).

## Spotlight Rules

An openly-governed, well-maintained build of the Spectral API linter:

| Project | What it is |
| --- | --- |
| [**spotlight-cli**](https://github.com/api-commons/spotlight-cli) | The linter + engine — OpenAPI, AsyncAPI, Arazzo. `npm i -g @spotlight-rules/spotlight-cli` |
| [**spotlight-spec**](https://github.com/api-commons/spotlight-spec) | Standalone ruleset specification + JSON Schema. |
| [**spotlight-vscode**](https://github.com/api-commons/spotlight-vscode) | The VS Code extension. |
| [**spotlight-rules**](https://github.com/api-commons/spotlight-rules) | The website — [spotlight-rules.com](https://spotlight-rules.com). |

Spotlight is a 1.0 rebrand of the open-source [Stoplight Spectral](https://github.com/stoplightio/spectral)
codebase (Apache-2.0).

> Naming map: GitHub org **api-commons** · npm scope **@spotlight-rules** · site **spotlight-rules.com**.
